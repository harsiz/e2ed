e2ed

End to end encryption for Discord DMs and server channels.

Open any channel, click the lock icon in the message toolbar, and enter a password both of you agreed on somewhere else (in person, over the phone, etc). e2ed never sends that password anywhere. The other person needs e2ed installed too, with the same password set for that conversation.

On Off No password

Click the lock again any time to turn encryption off or on for a conversation without losing the password. Text messages only. Files and attachments are not encrypted.

Passwords are saved per conversation in this browser so you do not have to re-enter them after a reload. They are never sent anywhere.
